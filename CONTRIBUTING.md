# Contributing to Home Assistant JSON Schemas
Thank you for your interest in contributing to this project. Even small improvements or bug reports are highly appreciated.

## How to contribute

### Reporting Issues

If you encounter any issues with the generated schemas (e.g., incorrect validation, missing properties, or errors in the CI pipeline), please open an issue in this repository.

* Please include the version of Home Assistant you are using.
* Describe the expected vs. actual behavior.

### Suggesting Improvements

If you have ideas on how to improve the schema generation process or the CI/CD pipeline:

1. Fork the repository.
2. Create your feature branch (git checkout -b feature/your-feature-name).
3. Commit your changes.
4. Push to the branch.
5. Open a Pull Request.
